# growi-outline-theme

GROWI v7 theme plugin inspired by [Outline wiki](https://www.getoutline.com)'s clean, minimal design language.

## Design Philosophy

This theme brings Outline's refined typography, subtle color palette, and spacious layout to GROWI. Key design characteristics:

- **Typography**: Noto Sans JP (和文) + Inter (欧文) font stack, optimized letter-spacing and font features
- **Color palette**: Outline's signature warm greys, blue accent (#0366d6), and carefully balanced dark mode
- **Sidebar**: Warm grey background (#EDF2F7 light / #08090C dark) matching Outline's sidebar
- **Cards & surfaces**: Subtle shadows, clean borders, elevated on hover
- **Form controls**: Warm grey input backgrounds, blue focus rings
- **Code blocks**: Smoke-tinted backgrounds with subtle borders
